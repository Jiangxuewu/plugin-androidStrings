# ExportAndroidStrings

这是一个用于导出 Android 字符串的项目。

**创建者**: Gemini-cli

## 简介

这是一个 Android Studio 插件，旨在帮助用户方便地导出其 Android 项目中所有语言下的字符串资源。用户在 Android Studio 中安装该插件后，即可轻松导出项目中的所有字符串。

## 功能

核心功能是导出 Android 项目中所有语言下的字符串。
